Backend Configuration
^^^^^^^^^^^^^^^^^^^^^

TypoScript influences many aspects of a TYPO3 site. For example, if
TypoScript is used in the User/Usergroup TypoScript field or in the
page TypoScript field, it will change the look and behavior of forms
in the backend.

The frontend rendering is defined by the TypoScript in the TypoScript
template. This document only covers frontend rendering and only with
TypoScript in general.
