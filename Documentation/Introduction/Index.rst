.. _about:

About this document
^^^^^^^^^^^^^^^^^^^

This document is meant to give a short introduction to how TypoScript
works and what TypoScript really is. It shall help to give a profound
understanding of the code, instead of just supplying snippets for
copying and pasting.


.. _what-s-new:

What's new
^^^^^^^^^^

This document is the "published" version of the wiki page
`http://wiki.typo3.org/Ts45min <http://wiki.typo3.org/Ts45min>`_ . All
examples should work with the latest stable release branch, TYPO3
version 4.4.
